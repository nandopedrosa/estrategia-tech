Introdução a Python

Python é uma das linguagens de programação mais populares e amplamente utilizadas atualmente. Por ser uma linguagem de programação de propósito geral, pode ser usada para uma variedade de aplicações e não se limita a nenhum campo específico.

Público-alvo

Observação: este curso não fornecerá um guia abrangente sobre o Python, mas fornecerá conhecimento suficiente para permitir que você comece a usar o Python.

Objetivos do curso

Os objetivos do curso são:

Números

Em Python, números são um dos tipos de dados fundamentais e representam valores numéricos. Existem três tipos de números em Python: inteiros (int), números de ponto flutuante (float) e números complexos (complex). Inteiros são números inteiros sem parte decimal, como 5, -8 ou 1000. Números de ponto flutuante têm uma parte decimal, como 3.14 ou -0.5. Números complexos são da forma a+bj, onde a e b são números reais e j é a unidade imaginária. Em Python, os números podem ser usados em operações matemáticas, como adição, subtração, multiplicação e divisão. Além disso, existem várias funções e bibliotecas disponíveis em Python para realizar operações matemáticas mais avançadas com números.

Convenção: nomes de variáveis começam com letra minúscula. Para nomes de variáveis com várias palavras, a maioria dos programadores usa sublinhados entre as palavras, mas alguns usam “maiúsculas”.

Strings

Uma string é uma sequência de caracteres entre aspas.

Em Python, você pode usar aspas simples ou duplas para strings. Você também pode inserir aspas simples dentro de strings com aspas duplas e vice-versa.

Python geralmente usa aspas simples ao imprimir resultados.

Não existe o tipo de dado "caractere" em Python.

Concatenação

Em Python, strings podem ser concatenadas (adicionadas) com o operador +.

Você também pode simplesmente deixar de fora o sinal + entre strings literais (entre aspas) e separá-las usando espaço em branco (espaços, tabulações ou novas linhas).
